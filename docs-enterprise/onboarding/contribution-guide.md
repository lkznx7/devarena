# Contribution Guide

## Process

1. Fork repository
2. Create feature branch
3. Make changes
4. Write tests
5. Submit PR
6. Code review
7. Merge

## Standards

- Follow [Coding Standards](../engineering/coding-standards.md)
- Use [Git Workflow](../engineering/git-workflow.md)
- Write tests
- Update documentation

## PR Requirements

- [ ] Tests passing
- [ ] No lint errors
- [ ] Description with context
- [ ] Related issue linked
