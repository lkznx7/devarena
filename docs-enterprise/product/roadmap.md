# Product Roadmap

## MVP Phases

### Phase 1: Foundation (Weeks 1-4)
- Infrastructure setup
- Authentication (JWT + OAuth)
- User profile & stats
- XP & streak system

### Phase 2: Core Features (Weeks 5-8)
- Challenge system
- Code execution (Piston)
- PvP matchmaking
- Real-time WebSocket

### Phase 3: Polish (Weeks 9-12)
- AI Mentor (Groq)
- Notifications
- Rankings
- Onboarding flow

## Post-MVP

- Guilds
- Battle Pass
- Mobile app
- Tournament system
