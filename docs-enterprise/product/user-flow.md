# User Flow

## Auth Flow

```
Landing → Register/Login → OAuth/JWT
                              ↓
                          Dashboard
```

## Challenge Flow

```
Browse Topics → Select Challenge → Read Description
      ↓
Write Code → Run Tests → Submit → XP Awarded
```

## PvP Flow

```
Enter Queue → Matchmaking → Match Found (WS)
      ↓
Code Submission → Timer End → Winner Declared
      ↓
XP + MMR Update → Result Screen
```

## Onboarding Flow

1. Welcome screen
2. Level assessment (5-8 questions)
3. Topic selection
4. First guided challenge
5. Reward (XP + badge)
