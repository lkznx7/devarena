# Gamification

## XP System

| Action | XP |
|--------|-----|
| Complete challenge | 100+ |
| PvP win | 50-100 |
| Daily login | 10 |
| Mission complete | 50-200 |

## Levels

- Level up every 1000 XP
- Topic-specific levels
- Visual progress bars

## Streak

- Daily activity required
- Streak freeze for Pro users
- Bonus XP on milestones

## Badges

| Category | Examples |
|----------|----------|
| Challenges | First solve, 100 solves |
| PvP | First win, 50 wins |
| Streak | 7-day, 30-day |
| Social | First follow |

## Rarity

Common, Uncommon, Rare, Epic, Legendary
